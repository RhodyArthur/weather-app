
## BONUS SECTION: Build a Weather App (10 points)

### Create a Command-Line Weather Application

Build a CLI application that fetches weather data from a public API.

**Use OpenWeatherMap API (free tier):**

- Sign up at: https://openweathermap.org/api
- Get free API key
- Documentation: https://openweathermap.org/current

**Bonus Features (Extra Credit):**

- Save favorite cities to a file
- Temperature unit conversion (Celsius/Fahrenheit)
- Color-coded output based on temperature
- Cache recent requests to avoid API rate limits

---

## Scoring Guide

- **80-100 points**: Excellent! Ready for Databases and FastAPI
- **60-79 points**: Good understanding, practice more with APIs
- **40-59 points**: Decent foundation, review HTTP and REST concepts
- **Below 40 points**: Review web fundamentals before proceeding

---

## Important Notes

- **API Rate Limits:** Be respectful of API rate limits, especially with GitHub (60 requests/hour without auth)
- **API Keys:** Never commit API keys to version control
- **Error Handling:** Always handle network errors gracefully
- **Timeouts:** Always set timeouts for requests
